# Planejamento-estrat-gico-de-TI
O *Planejamento Estratégico de TI* é essencial para que a tecnologia atue como um *fator estratégico*, apoiando o crescimento sustentável da organização.  
Quando bem aplicado, ele *melhora o desempenho, **reduz riscos* e garante que a *TI acompanhe as mudanças do mercado* e as *necessidades do negócio*.  

Na *aula de hoje, aprendemos que o planejamento estratégico de TI é fundamental para o **alinhamento entre tecnologia e objetivos organizacionais*, contribuindo para uma gestão mais eficiente, segura e competitiva.
